About Task Pilot

Welcome to Task Pilot 🎉 — a simple yet powerful task management tool built with Flask and SQLite. This app helps you stay organized by letting you:

  • 🔹 User authentication (login, register, logout)
  • 🔹 Add new tasks to keep track of what needs to be done.
  • 🔹 Prioritizing tasks for better management.
  • 🔹Update task status (mark as complete or incomplete).
  • 🔹 Delete tasks you no longer need.
  • 🔹 Secure authentication, so each user has their own private todo list.
